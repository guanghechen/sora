Common type definitions for @guanghechen packages.

## Install

- npm

  ```bash
  npm install --save-dev @guanghechen/types
  ```

- yarn

  ```bash
  yarn add --dev @guanghechen/types
  ```

## Usage

```typescript
import type {
  // Disposable types
  IDisposable,
  IBatchDisposable,
  // Error types
  ErrorLevelEnum,
  ISoraError,
  ISoraErrorCollector,
  // Path types
  IPathResolver,
  IPathResolverParams,
  IWorkspacePathResolver,
} from '@guanghechen/types'
```
